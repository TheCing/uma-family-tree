# frontend

The React + TypeScript + Vite SPA for **Paddock** (an Uma Musume pedigree planner).

This is the project's only npm workspace. See the [root README](../README.md)
for an overview, setup, build, and deployment instructions.

```bash
npm install      # from the repo root
npm run dev      # from the repo root — starts Vite on http://localhost:5173
```

Workspace scripts (run from this directory): `dev`, `build`, `lint`, `format`,
`preview`, `clean`.
